# Arrow

这是一个 Unity 箭头消除游戏原型，对应“箭头消除王”的核心玩法。

## 玩法规则

- 默认从第 3 关开始。
- 关卡初始生命值为 3。
- 关卡限定时间为 8 分钟。
- 场景中会生成大量不重叠的箭头。
- 箭头由多个网格段组成，可以很短，也可以很长。
- 长箭头允许在网格中蜿蜒，但整根箭头始终只会朝箭头头部指向的方向移动。
- 点击箭头时，会先预测它沿自身方向移出场景的路径。
- 如果路径上没有其他箭头阻挡，箭头会整体滑出场景并被移除。
- 如果前方被其他箭头阻挡，箭头会向前顶一下后回弹到原位置，并扣除玩家 1 点生命值。
- 当所有箭头都被移除时，关卡胜利。
- 当生命值归零或倒计时结束时，关卡失败。

## 道具

- 橡皮擦：点击道具后进入点选模式，再点击任意箭头即可直接擦除该箭头。
- 魔法棒：立即随机消除最多 3 根箭头。
- 时间冻结器：当前实现为给关卡倒计时增加 60 秒。

## 运行入口

- `GameInitializer`：启动时创建必要的管理器、UI 根节点和箭头生成器。
- `LevelManager`：管理关卡配置，默认第 3 关，3 点生命值，8 分钟倒计时。
- `GameManager`：管理生命、时间、胜负状态、箭头列表和输入锁。
- `ArrowGenerator`：按网格生成不重叠箭头，支持短箭头和少量长蜿蜒箭头。
- `Arrow`：处理点击、移动路径预测、出界移除、阻挡回弹和碰撞占位。
- `ItemManager`：处理橡皮擦、魔法棒、时间冻结器三个道具。
- `UIManager`：自动绑定现有 UI prefab 中的按钮和文本，刷新生命、时间、道具数量和胜负面板。

## 本地验证

当前机器没有安装 Unity Editor，也没有可用的 C# 编译器，所以还不能在本机完成 Unity 导入编译验证。

已完成的本地验证：

- `git diff --check` 通过。
- 使用与 `LevelManager` 相同参数做了 50 轮玩法仿真。
- 每轮都生成满 72 根箭头。
- 所有仿真棋盘均通过“不重叠”检查。
- 平均每根箭头约 5.79 个网格段。
- 仿真中覆盖了可直接移出场景和被阻挡回弹两类移动结果。
- 道具仿真通过：橡皮擦移除 1 根，魔法棒移除 3 根，时间道具将 480 秒增加到 540 秒。
- prefab/脚本一致性检查通过：自定义脚本 GUID 都能找到对应 `.cs.meta`。
- `SceneEditorWindow` 已使用 `UNITY_EDITOR` 包裹，避免运行时构建引用 `UnityEditor`。

仍建议在安装 Unity Editor 的环境中继续验证：

- 首次导入后的 C# 编译结果。
- prefab 序列化字段是否按 Inspector 预期刷新。
- 箭头 sprite 的实际缩放、点击碰撞手感和 UI 布局表现。
- 真机或目标平台上的输入、音效和倒计时表现。
